      初识OpenResty

      Post on 2016-06-18

      OpenResty介绍

      OpenResty ™ 是一个基于 Nginx 与 Lua 的高性能 Web 平台，其内部集成了大量精良的 Lua 库、第三方模块以及大多数的依赖项。用于方便地搭建能够处理超高并发、扩展性极高的动态 Web 应用、Web 服务和动态网关。

      安装所需扩展

      yum -y install readline-devel pcre-devel openssl-devel

      下载

      wget https://openresty.org/download/ngx_openresty-1.9.7.1.tar.gz

      解压

      tar zxvf ngx_openresty-1.7.2.1.tar.gz

      编译安装

      ./configure        --with-luajit\                                            
--without-http_redis2_module    \                                
--with-http_iconv_module

gmake && gmake install

      启动

      /usr/local/openresty/nginx/sbin/nginx  -c /usr/local/openresty/nginx/conf/nginx.conf -p /usr/local/openresty/nginx/

      测试

      访问你的服务器查看是否有Ngixn欢迎页面有代表你成功了

      输出第一个hello world

      vim /usr/local/openresty/nginx/conf/nginx.conf

      加入以下代码

      location /test/ {
      default_type text/html;
      content_by_lua '
          ngx.say("<p>hello, world</p>")
      ';
  }

      重新加载配置文件

      /usr/local/openresty/nginx/sbin/nginx -c /usr/local/openresty/nginx/conf/nginx.conf -p /usr/local/openresty/nginx/ -s reload

      访问服务器的/test/会看见
      hello, world

      引用外部lua文件

      将刚才插入ngxin的代码更换成以下这样

      location /hello {

       content_by_lua_file lua/hello.lua;

   }

      创建lua目录并vim hello.lua 加入以下代码
      ngx.say(“

      hello, world!!!

      “)

      重新加载配置文件

      usr/local/openresty/nginx/sbin/nginx -c /usr/local/openresty/nginx/conf/nginx.conf -p /usr/local/openresty/nginx/ -s reload

      访问服务器的/test/会看见
      hello, world
