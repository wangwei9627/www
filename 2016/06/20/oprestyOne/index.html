      OpenResty入门

      Post on 2016-06-20

      OpenResty第一个小实例

      插入ngxin的以下代码

      location /getstr {

       content_by_lua_file lua/getstr.lua;

   }

      在lua目录下面vim getstr.lua 加入以下代码

      local args = ngx.req.get_uri_args()  
local salt = args.salt
if not salt then
ngx.exit(ngx.HTTP_BAD_REQUEST)
end

local string = ngx.md5(ngx.time() .. salt)
ngx.say(string)

      重新加载配置文件

      usr/local/openresty/nginx/sbin/nginx -c /usr/local/openresty/nginx/conf/nginx.conf -p /usr/local/openresty/nginx/ -s reload

      访问服务器的/test/会看见
      md5的字符串

      不需要每次都重启的设定

      参考文档

      https://github.com/openresty/lua-nginx-module#lua_code_cache

      nginx.conf 文件中加入 lua_code_cache off;

      context: http, server, location, location if(可用在以上位置)

      重新加载配置文件

      看见这样一行提示说明你成功了

      nginx: [alert] lua_code_cache is off; this will hurt performance in /usr/local/openresty/nginx/conf/nginx.conf:38

      注意只适用于本地开发。生产环境不建议用会大大影响效率！！！
