      shell介绍与定义变量

      Post on 2016-07-01

      学习shell需要的基础知识储备

      1. vi/vim编辑器命令，vimrc设置
      2. 基础命令
      3. 一般高端的网络服务

      shell简介

      shell翻译过来就是贝壳的意思。当命令不在命令行执行，而是通过一个程序文件执行的时候，该程序就是shell.

      shell脚本建立和执行

      shell 脚本（bash shell 程序）通常实在编辑器（vi / vim）中编写，由Unix/Linux命令bash shell命令，程序机构控制语句和注释等内容组成。

      脚本开头

      #！/bin/bash

      或者

      #！/bin/sh

      ‘#!’又称为幻数，在执行bash脚本的时候，内核会根据它来确定该用那个程序来解释脚本中的内容，这一行必须在脚本顶端第一行，否则就是注释了,因为sh链接到bash，所以建议使用bash

      注意，因为linux 默认是bash执行，所以就算不加开头也可以正确执行，但这样不规范，也容易出错。如果开头不指定解释器，那么要用对应的解释器来执行脚本，例如

      sh test.sh

      注释 一般就是用#

      shell 程序在运维工作中的作用地位？

      shell 脚本很擅长处理纯文本类型的数据， 而linux中几乎所有的配置文件，日志文件等都是纯文本的文件，隐藏，如果学好了shell脚本语言，就可以在lunux下发挥巨大的作用

      shell脚本语言是肉类型脚本语言

      shell与其他脚本语言的优势

      shell 的优势在于处理操作系统底层的业务（大量的命令作支撑），一键安装，报警脚本，常规的业务应用，shell开发起来简单迅速！

      第一个shell脚本

      #!/bin/bash
echo "Hello World !"

      运行Shell脚本有两种方法：

      1作为可执行程序

      chmod +x ./test.sh  #使脚本具有执行权限
./test.sh  #执行脚本

      2作为解释器参数

      /bin/sh test.sh
/bin/php test.php

      定义变量

      定义变量时，变量名不加美元符号（$），如：

      variableName="value"

      注意，变量名和等号之间不能有空格，这可能和你熟悉的所有编程语言都不一样。同时，变量名的命名须遵循如下规则：

      首个字符必须为字母（a-z，A-Z）。
      中间不能有空格，可以使用下划线（_）。
      不能使用标点符号。
      不能使用bash里的关键字（可用help命令查看保留关键字）。

      变量定义举例：

      myUrl="http://see.xidian.edu.cn/cpp/linux/"
myNum=100

      使用变量

      your_name="xxx"
echo $your_name
echo ${your_name}

      变量名外面的花括号是可选的，加不加都行,推荐给所有变量加上花括号，这是个好的编程习惯
      使用 readonly 命令可以将变量定义为只读变量，只读变量的值不能被改变
      使用 unset 命令可以删除变量
